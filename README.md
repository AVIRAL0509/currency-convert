# Currency Converter

A simple and responsive web-based currency converter that allows users to convert amounts between different currencies using live exchange rates.

## Features
- Convert between major currencies (USD, EUR, INR, GBP, JPY, AUD)
- Real-time exchange rate fetching
- Input validation and error handling
- Clean and minimal UI
- Fully responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Public Exchange Rate API
